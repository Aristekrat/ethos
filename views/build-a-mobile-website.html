Responsive Design: the Cheap and Easy Mobile Solution

Most laypeople think they have to build a seperate mobile site if they want a site that will work with phones. This approach makes sense if your site receives 100k+ in mobile traffic. Otherwise, it's far more expensive than a better option: Responsive Design.

Responsive design allows your site to intelligently adapt based on the device accessing it. If it's a phone, it shrinks. If it's a massive monitor, it expands. Best of all, it only requires modifications to your site's stylesheet, not a completely new website.
